# String utils

Utilities to handle strings.

### Version
0.0.1
### Installation

Use npm to install:

```sh
$ npm install -g string-utils
```
### Example
First import string-utils with require:
```javascript
const sutils = require('string-utils');
```
Getting words from string
```javascript
let words = sutils.words('lorem ipsum dolor');
//['lorem', 'ipsum', 'dolor']
```
